# DEMO-WSO2-MI-DEPLOYMENT-CICD

This repository contains deployment artifacts for WSO2 Micro Integrator (MI) as part of the Continuous Deployment (CD) process in the CI/CD demo.

## Purpose
The repository is designed to showcase the CD process for WSO2 MI CAR applications in a development environment. It serves as a central location for managing deployment configurations and artifacts used in the pipeline.

## Contents
The repository includes:
- **CAR Artifacts**: WSO2 MI Composite Application Archive (CAR) files used for deployments.

## Key Features
1. Demonstrates Continuous Deployment for WSO2 MI.
2. Includes sample CAR applications to highlight the deployment flow.
